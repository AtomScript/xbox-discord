# xbox-profile
With this piece you can view an Xbox Live user's data by image. Use this mini library for your discord bot 

## xbox.js
Use this as a command.

## proPrint.js
Responsible for creating the image.

# Commands


## find
Example: <prefix>xbox find DiegoGamer400
